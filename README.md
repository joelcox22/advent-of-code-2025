# Advent of Code 2025 Solutions

https://adventofcode.com/

Requirements: [install deno](https://docs.deno.com/runtime/getting_started/installation/)

---

To run solutions against sample inputs (`sample.txt`):

- cd to folder for the relevant day
- run `./main.ts`

To run against real inputs:

- cd to folder for the relevant day
- save your puzzle input to `input.txt`
- run `./main.ts input.txt`
